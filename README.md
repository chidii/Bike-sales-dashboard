# Bike-sales-dashboard
From Raw Customer Data to Business Insights: Building an Interactive Bike Sales Dashboard in Microsoft Excel

1. Project Overview
This project demonstrates how Microsoft Excel can be used to transform raw customer data into actionable business insights through data cleaning, exploratory data analysis and interactive dashboard design.

 Acting as a data analyst for a fictional bicycle retailer, I analyzed customer demographic and purchasing data to identify the factors influencing bike purchases. The goal was to uncover meaningful trends that could support data-driven marketing and business decisions.

2. Business Problem
The company had collected customer information but lacked visibility into the characteristics of customers who were most likely to purchase a bike. This analysis aimed to answer the following business questions:

- Which customer segments are most likely to purchase a bike?
- Does income influence purchasing behaviour?
- How does age affect purchasing decisions?
- Does commute distance impact bike purchases?
- Are there regional differences in customer buying patterns?

3. Dataset
The dataset contains customer demographic and purchasing information, including:

- Customer ID
- Gender
- Marital Status
- Age
- Education
- Occupation
- Annual Income
- Commute Distance
- Region
- Home Ownership
- Number of Cars
- Number of Children
- Bike Purchase (Yes/No)

5. Data Cleaning & Preparation
To improve data quality before analysis, I performed the following tasks:

- Removed duplicate records
- Standardized categorical values for consistency
- Reviewed and corrected formatting issues
- Created customer age brackets using Excel formulas
- Prepared a clean working dataset for analysis

6. Exploratory Data Analysis
Using PivotTables, I explored customer purchasing behaviour across multiple dimensions, including:

- Average Income vs Bike Purchases
- Age Bracket Analysis
- Gender Distribution
- Commute Distance Analysis
- Regional Performance

These analyses provided a deeper understanding of customer demographics and purchasing patterns.

7. Dashboard Features
The interactive Excel dashboard allows users to explore customer purchasing trends using filters and visualizations. It includes

- Average Income by Bike Purchase
- Bike Purchases by Age Bracket
- Bike Purchases by Commute Distance
- Interactive slicers for: Marital Status, Region and Education

8. Key Insights

The analysis revealed several important business insights:

- Customers with higher average incomes were generally more likely to purchase bikes.
- Middle-aged customers represented the strongest purchasing segment.
- Customers with moderate commute distances showed higher purchase rates than those with very short or very long commutes.
- Regional differences suggest opportunities for more targeted marketing strategies.

---

9. Business Recommendations

Based on the analysis, I recommend that the business:

- Focus marketing campaigns on higher-income customer segments.
- Increase promotional efforts toward middle-aged professionals.
- Develop region-specific marketing strategies based on purchasing trends.
- Create targeted campaigns for customers with moderate commuting distances.
